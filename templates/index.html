Register a Computational Asset

Use this form to prepare and submit metadata for a Computational Asset through the aiondemand Python SDK. The asset itself should remain hosted on a stable external source such as a repository, landing page, or archive.

Recommended workflow
  1. Find or enter the related Project identifier.
  2. Complete the required metadata fields.
  3. Preview the JSON payload.
  4. Submit the asset to AIoD.
  5. Verify the returned identifier.
